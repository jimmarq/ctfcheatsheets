Methods for strings (a = CoW)
.lower() #returns lowercase version of string (ex: "a.lower()" Output: cow)
.upper() #returns uppercase version of string (ex: "a.upper()" Output: COW)
.strip() #removes blank spaces from beginning and end of string (ex[a = '   Cow  ']: "a.strip()" Output: 'Cow')
.replace() #replaces given string with another (ex: "a.replace('c','h')" Output: how)
.split() #splits a string into a list (ex[a=Good Morning]:"a.split(' ')" Output: ['Good', 'Morning'])
